Fashion Mart

A full-stack fashion retail web application with product browsing, cart, checkout, order tracking, delivery management, and admin dashboard.

View Source Code

Project Features

Product Catalog

Browse fashion products with categories, search, price, stock, and ratings.

Shopping Cart

Add products to cart, update quantity, calculate totals, and checkout.

Order Tracking

Customers can view order history, order status, delivery status, and payment status.

Admin Dashboard

Admin and delivery management features for a retail workflow.

Note: This GitHub Pages version is only a static project showcase. The full app requires Flask hosting and a MySQL database.
